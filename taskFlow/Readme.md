# 🚀 TaskFlow – Smart Task Manager

## 📌 Project Overview

**TaskFlow** is a modern and interactive Smart Task Manager built using **Vanilla JavaScript (Basic → Advanced concepts)**.
The goal of this project is to simulate a real-world productivity tool where users can manage their daily tasks efficiently using a clean and intuitive interface.

This project evolves from a simple **To-Do App** into a more advanced **Kanban-style task management system**.

---

## 🎯 Project Goal

The main objectives of TaskFlow are:

* Apply **core to advanced JavaScript concepts** in a real-world project
* Build a **scalable and maintainable frontend architecture**
* Understand **state management without frameworks**
* Prepare for transitioning into **React & Full Stack (MERN)** development
* Create a **portfolio-ready project** for interviews

---

## 🧠 Key Concepts Used

* DOM Manipulation
* Event Handling
* Event Delegation
* Closures (for better state control)
* Debouncing (for search optimization)
* LocalStorage (data persistence)
* Modular JavaScript (separation of concerns)

## ⚙️ How the Application Works

1. User enters a task in the input field
2. Task is stored in a **JavaScript array (state)**
3. UI updates dynamically using render functions
4. Events are handled centrally (click, keypress, etc.)
5. Data is saved in **LocalStorage** for persistence
6. Tasks can be managed (add, delete, update, move)
